## Getting Started
 
Constructor is a tool maintained by the team, it helps creating newsletter campaigns.

<Steps>
### Install [Bun](https://bun.com/docs/installation#windows)
 
Open powershell and run the following command to install Bun:

```powershell copy
powershell -c "irm bun.sh/install.ps1|iex"
```

Restart your terminal or computer to apply the changes.

Check if bun is installed correctly by running:

```powershell copy
bun --version
# Should return something like: 1.4.0 (or the latest version)
```

<Callout type="important">
  If you’ve installed Bun but are seeing a command not found error, you may have
  to manually add the installation directory (~/.bun/bin) to your PATH.
</Callout>

Follow the instructions in the [Bun documentation](https://bun.com/docs/installation#add-bun-to-your-path) to add Bun to your PATH.

### Clone the repository

<Callout type="info">
  Git should be already preinstalled on your computer, if not, you can install
  it from [here](https://git-scm.com/downloads).
</Callout>

<Callout type="important">
  Make sure to replace `<dir>` with the actual directory name where you want to clone the repository.
  If you want to clone it in the current directory, you can simply use `.` instead of `<dir>`.
</Callout>

```powershell copy
git clone https://github.com/Beliani-HTMLTeam/constructor.git <dir>
```

### Install dependencies

After successfully cloning the repository, navigate to the project directory and install the dependencies using Bun:

```powershell copy
bun install
```

First time installation may take a while, so please be patient. Later installations will be faster as Bun caches the dependencies.

### Create Google Cloud Service Account

1. To access Google Sheets API, you need to create a Google Cloud Service Account and generate a private key.<br/>https://console.cloud.google.com/projectcreate

2. After creating the project, enable Google Sheets API<br/>https://console.cloud.google.com/apis/library/sheets.googleapis.com

3. Go to "**IAM & Admin**" > "**Service Accounts**" and click "**Create Service Account**".<br/>Follow the instructions on screen to create a service account.<br/>

4. Download JSON key from "**IAM & Admin**" > "**Service Accounts**" > _your service_ > "**Keys**" > "**Add Key**" > "**Create new key**" > "**JSON**".
